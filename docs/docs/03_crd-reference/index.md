# CRD Reference

The Glasskube Operator can be used to manage a variety of applications.
The list of supported applications is constantly growing, but currently consists of:

- [Gitea](gitea)
- [GitLab](gitlab)
- [GlitchTip](glitchtip)
- [Keycloak](keycloak)
- [Matomo](matomo)
- [Nextcloud](nextcloud)
- [Odoo](odoo)
- [Plane](plane)
- [Vault](vault)
